# sde1junaidkhan

Junaid Khan's professional resume as an npm package.

## Installation & Usage

You can run my resume directly without installing:

```bash
npx @sde1junaidkhan
```

Or install globally:

```bash
npm install -g sde1junaidkhan
@sde1junaidkhan
```
